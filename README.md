## Description: 
A very sumplified version of Slack Chat. Here you can:
* login users and create new ones:
* create, rename and remove channels
* and, you know, chat :)

## Tech Stack:
* React
* React Router DOM
* React Bootstrap
* React i18next
* React Toastify 
* Redux Toolkit
* Formik
* Yup
* Axios
* Leo Profanity

## Installation:
  ### Requirements:
    - Mac / Linux
    - Node.js [>=16.x]

  ### Dependencies:
    make install

  ### Build (and dependencies also):
    make build

  ### Server Start:
    make start

  ### Lint:
    make lint-frontend
